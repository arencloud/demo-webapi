# API Documentation

## OpenAPI source

The API entity references this OpenAPI file:

- `docs/swagger.yaml`

In Red Hat Developer Hub, open the `demo-webapi-api` entity to view the rendered API schema.

## Endpoint summary

### `GET /api/ping`

Returns ping response and request headers.

### `GET /realms/ext/npwl-user/logout`

Returns logout response, request headers, and optional `extranetUrl` query parameter.

### `GET /realms/ext/protocol/cas/connect`

Returns connect response, request headers, and optional query parameters `extranetId` and `p`.
